# Voice Command to Text using Wav2Vec2

## Project Overview

This project converts spoken voice commands into text using a deep learning speech recognition model. The system takes an audio input and processes it using the Wav2Vec2 model to generate the corresponding textual output.

## Features

* Converts voice commands into text
* Uses a pretrained Wav2Vec2 speech recognition model
* Supports audio file input
* Fast and accurate speech transcription

## Technologies Used

* Python
* Wav2Vec2 Speech Recognition Model
* PyTorch
* Transformers Library

## How It Works

1. The user provides an audio input containing voice commands.
2. The audio is preprocessed and converted into the required format.
3. The Wav2Vec2 model processes the audio waveform.
4. The model outputs the predicted text transcription.

## Installation

Clone the repository:

git clone <your-repository-link>

Install the required dependencies:

pip install -r requirements.txt

## Usage

Run the main script:

python app.py

Provide an audio input file, and the system will convert the speech into text.
